# Challenge Rate Limiter

## Instalation
1. Configurar la versión adecuada de NodeJS
```
nvm use
```
2. Instañar dependencias
```
nvm install
```
3. Levantar la aplicacion localmente
```
nvm run dev
```

## Examples
```curl
curl -H "User-Id: 1" -X GET "http://localhost:4000/"
```
